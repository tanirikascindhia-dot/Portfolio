The problem

Alstom's internal teams relied on manual SharePoint document transfers and disconnected reporting, which made data slow to move, hard to trust, and even harder to turn into a dashboard leadership could act on.

Outcome

A working, end-to-end automation ecosystem: documents that used to move by hand now flow through scripted migration, a governed Lakehouse, and Power BI dashboards leadership can actually open and trust — with a validated Translation Portal alongside it.

What I did

  • SharePoint migration — executed document migration from one environment to another using PowerShell and JSON scripts.
  • Workflow automation — developed automated workflows using Power Automate and Power Apps to streamline internal business processes.
  • Reporting — generated reports for data tracking and validation using Excel.
  • Lakehouse build — built a Lakehouse solution in Microsoft Fabric, integrating data warehousing and data lake capabilities with efficient data ingestion pipelines.
  • Data engineering — developed scalable data processing workflows using Apache Spark, PySpark, and T-SQL for data transformation and analysis.
  • Dashboards — built interactive Power BI dashboards within Microsoft Fabric, leveraging unified Lakehouse data.
  • Archiving — created archives of historical projects in Azure Storage Explorer.
  • UX validation — conducted UI/UX testing and validation of the Translation Portal.
  • AI exploration — explored AI Builder for AI-powered document processing.

Case 03 · Personal Project

Smart Laser Security System, built on a microcontroller.

A laser-grid intrusion detection system for home and commercial spaces — break the beam, trigger the alert.

NodeMCU ESP8266C++IoT

What I built

Developed a microcontroller-based intrusion detection system using NodeMCU ESP8266 and C++, with a laser-and-photoresistor grid that detects a break in the beam path.

Responsibilities

Programmed real-time alert functionality end-to-end — from sensor read, to interrupt logic on the microcontroller, to triggering the notification.

Use case

Designed for home and commercial security monitoring — a low-cost, always-on layer that flags unauthorized entry the moment a beam is crossed.
